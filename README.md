Essentials
===========

This is a long-term plugin that aims to port most used Essentials commands to MCServer. Commands already implemented in Core won't be implemented.

/more

/spawnmob

/biome

/repair

/feed

/heal

/enchant

/xp

/ping

/burn

/vanish

/hat
